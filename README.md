# MX Ergo S ジェスチャー・チートシート・オーバーレイ

MX Ergo S のジェスチャーボタンを押している間、現在アクティブなアプリケーションに応じたジェスチャーアクションのチートシートを画面にオーバーレイ表示する Windows 常駐アプリです。

## セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. Logi Options+ の自動読み込み

このアプリケーションは、Logi Options+ の設定ファイル (`%LOCALAPPDATA%\\LogiOptionsPlus\\settings.db`) を自動的に読み込みます。手動での設定ファイルの作成や更新は不要です。

Logi Options+ アプリケーションで設定を変更した場合は、システムトレイのメニューから「settings.db 再読み込み」を実行することで変更が反映されます。

## 起動

最も簡単な起動方法は、プロジェクトフォルダ内にある **`start_overlay.pyw`** をダブルクリックすることです。
（黒いターミナル画面が一切表示されず、完全にバックグラウンドで起動します）
※注意: 起動する前に、必ず `pip install -r requirements.txt` で必要なライブラリをインストールしておいてください（ライブラリがないと一瞬で閉じてしまいます）。

コマンドラインから起動する場合は以下の通りです：
```bash
python main.py
```

- 画面の右下にコントロールウィンドウが表示されます。
- 「▶ 表示する」ボタンを押すと、アクティブなウィンドウに応じたジェスチャー設定が画面にオーバーレイ表示されます。
- オーバーレイ表示中は、アクティブなアプリケーションを切り替えるたびに表示内容が自動で更新されます。
- 「■ 非表示にする」ボタンを押すとオーバーレイが消えます。

## システムトレイメニュー

- **settings.db 再読み込み**: Logi Options+ 側で行った設定変更を再読み込みして反映します。
- **終了**: アプリを終了します。
